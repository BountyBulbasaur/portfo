Blockchain

In order to fully understand the significance of a blockchain, I decided to learn how to code my own using Solidity. This helped me learn how blockchains work and get a basic "behind the scenes" view of solidity.

Solidity

I began learning Solidity with CryptoZombies, reading Mastering Ethereum, and practice from Chainlink's 2022 hackathon. This gave me the ability to really understand reading and writing Solidity code.

Secureum Bootcamp

I am grateful for the Secureum team for posting their videos for free on YouTube. I watched their videos and took note of the many best practices and auditing tools they presented so I can use them on the smart contracts I've written myslelf.

Staking Contracts

These contracts were a Chainlink hackathon exercise where contracts I created two contracts: one to stake tokens, and another to mint tokens that can be staked and rewarded for staking.

DeFi-Quant

This project was a way to practice writing scripts. This also helped deepen my understanding of DeFi by creating a script that can access the Aave lending pool.

DAO

This project was to create a DAO or a Decentralzed Autonomous Organization that is driven to make decisions with the use of governance tokens with a governor contract, and a time lock contract.

NFT

This NFT project was created so I can learn how NFTs work. Although I want to focus more on DeFi, this was a good project that can help me deepen my knowledge of Solidity and different types of Smart Contracts.
